# 📊 合约复盘教练

## 功能
逐笔分析合约交易，找出亏损根因，建立错误档案。

## 错误模式识别
| 错误类型 | 判断标准 | 建议 |
|---------|---------|------|
| 追涨开多 | 开仓价 > 近4h最高价90% | 等回调再入 |
| 杀跌开空 | 开仓价 < 近4h最低价110% | 等反弹再入 |
| 无止损 | 亏损超10%仍持仓 | 强制设置止损 |
| 情绪加仓 | 亏损后30分钟内加仓 | 冷静期60分钟 |
| 过度杠杆 | 杠杆 > 10倍 | 上限5倍 |

## 输出示例
```
📊 合约复盘报告 2026-03-04

本周交易：5笔 / 盈利：2笔 / 亏损：3笔
总盈亏：-40 USDT

❌ 高频错误（本周）：
1. 情绪加仓 × 2次（亏损占比68%）
2. 无止损持仓 × 1次

✅ 做对的事：
- BTC多单严格止盈，锁定+320U

🔧 本周改进重点：
强制止损 + 禁止连续亏损后30分钟内开仓
```
