# 🗺️ 收益地图子Agent

## 功能
实时扫描币安全生态理财产品，告诉用户同样的钱放哪里收益最高。

## 对比产品
| 产品 | 流动性 | 风险 | API |
|------|--------|------|-----|
| 活期理财 (Flexible) | 随时存取 | 极低 | 公开 |
| BNB Vault | 随时 | 低 | 公开 |
| Launchpool质押 | 锁定期 | 低 | 公开 |
| 定期理财 | 锁定 | 低 | 公开 |
| 简单赚币 | 活期/定期 | 中 | 公开 |

```

│ 500U → BNB Vault    年化 8.2%       │
│ 300U → Launchpool   年化 12.4%      │
│ 200U → 活期理财     年化 3.1%       │

└─────────────────────────────────────┘

└─────────────────────────────────────┘

vs 现在放着不动：年收益 0 USDT
你已经少赚了每天 ≈2.5 USDT 💸
```

## 数据接入
- 币安Earn产品列表：GET /sapi/v1/lending/union/productList
- 实时利率：公开接口，无需API Key
