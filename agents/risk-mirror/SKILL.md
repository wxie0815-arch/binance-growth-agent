# 🪞 风险镜子子Agent

## 功能
分析用户历史交易记录，生成「交易者人格报告」。

## 分析维度
1. **交易类型判断**: 趋势型 / 波段型 / 频繁短线型 / 情绪驱动型
2. **止损纪律**: 有无止损习惯，止损执行率
3. **追涨杀跌指数**: 高位买入频率 vs 低位买入频率
4. **持仓集中度**: 单币种持仓占比分布
5. **情绪化指数**: 大跌后加仓频率，连亏后急于翻本的次数

## 输出报告示例
```
🪞 你的交易者人格报告

类型：情绪驱动型交易者（占比78%）
致命弱点：追涨杀跌，缺乏止损纪律
优势：持仓分散，不重仓单一标的

详细得分：
- 止损纪律: 32/100 ⚠️
- 情绪控制: 41/100 ⚠️  
- 仓位管理: 67/100 ✅
- 入场时机: 55/100 📊

本周建议：先解锁收益地图，把闲置资金优化配置，
再针对止损纪律做苏格拉底训练。
```
