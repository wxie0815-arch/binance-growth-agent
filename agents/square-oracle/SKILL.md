## 动态评分模型

### Step 1: 判断今日市场情绪
```
从 D1 数据计算：
- BTC 24h 涨幅 > +3%  → 情绪 = "大涨"    市场系数 = 1.6
- BTC 24h 涨幅 > +1%  → 情绪 = "上涨"    市场系数 = 1.3
- BTC 24h 跌幅 < -3%  → 情绪 = "大跌"    市场系数 = 1.6（恐慌话题更热）
- BTC 24h 跌幅 < -1%  → 情绪 = "下跌"    市场系数 = 1.3
- 其余               → 情绪 = "震荡"    市场系数 = 1.0
```

### Step 2: 内容类型基础权重
```
话题类型                    基础权重    大涨/大跌时额外加成
────────────────────────────────────────────────────
行情分析（BTC/ETH趋势）       3.0         +1.0
新币/Launchpad/Launchpool    2.8         +0.5（新币不受行情影响）
币安官方活动解读              2.5         无
地缘政治影响分析              2.2         +0.8（暴跌时更热）
山寨季/板块轮动分析           2.0         +0.5
交易技巧/仓位管理             1.8         +0.3
AI×Web3 趋势                 1.5         无
Meme/热梗结合行情             1.5         +0.5
个人成长/交易心态             1.2         +0.2（暴跌时情绪共鸣）
日常观点/项目解读             1.0         无
```

### Step 3: 计算每个候选话题总分
```
话题总分 = 
  基础权重
  × 市场情绪系数
  × 时效性系数（6h内数据=1.5 / 12h内=1.2 / 24h内=1.0）
  × 官方关联系数（币安公告直接相关=1.8 / 间接=1.3 / 无关=1.0）
  × KOL讨论系数（3个以上KOL在聊=1.4 / 1-2个=1.2 / 无=1.0）
  × 竞争密度系数（已有大量文章=0.7 / 适中=1.0 / 稀缺话题=1.3）
```

### Step 4: 历史规律校正
对照下方「历史规律库」，如果话题符合高频上榜模式，额外 +0.3

## 历史规律库（实战总结）

### 🔴 必上榜情形（命中率 >80%）
1. **BTC 单日涨超 5%** → "这次牛市启动了吗" 类行情判断必爆
2. **新币首日上线币安** → Launchpool/Launchpad 分析当天最热
3. **美联储/美国 CPI 数据日** → 宏观+币价联动分析必上榜
4. **币安官方大活动** → 参与攻略类文章暴涨
5. **某代币单日涨超 30%** → 该代币解读文章秒上榜

### 🟡 高概率上榜（命中率 50-80%）
6. **周末行情** → 交易策略/持仓建议类话题涨幅 +50%
7. **市场剧烈震荡** → 恐慌还是机会？对立观点帖子互怼
8. **Meme 币季节** → 结合当下热梗的币种分析
9. **交易所集中提币** → 安全/资产保护类话题突然热门
10. **空投季** → 操作指南类文章持续稳定流量

### 🟢 稳定流量话题（长期有效）
11. 每周 BTC 行情复盘（周一发）
12. 月度持仓总结
13. 新手教程（持续有新人进场）
14. 交易心态/情绪管理（跌市高峰）
15. 币安功能使用技巧

### ❌ 避坑（容易没流量）
- 纯技术指标分析（太专业，受众窄）
- 无数据支撑的纯观点帖
- 标题不够吸引眼球（无数字、无悬念）
- 跟热点太慢 > 12小时后发同类话题
- 字数超过 800 字（阅读完成率下降）

## 文章生成模块（v3.0新增）

### 触发条件
- 触发词：「生成文章」「帮我写」「今日发帖」「自动发布」
- 每日定时任务：北京 08:00 / 12:00 / 20:00 三波

### 必发代币（每日强制）
每日文章中必须包含以下代币各至少1篇：
- ` $BTC `（比特币，关联永续合约）
- ` $ETH `（以太坊，关联永续合约）
- ` $XAU `（黄金，关联币安黄金永续）
- ` $XAG `（白银，关联币安白银永续）

### 每日发帖数量
- 最少：10篇
- 目标：12-15篇
- 分三波：早批4篇（08:00） / 午批4篇（12:00）/ 晚批4-7篇（20:00）

### 文章生成流程
```
1. 调用预言机获取今日热榜Top5话题
2. 按话题分配文章（必发代币优先保证）
3. 生成草稿
4. 过 humanizer-cn v3.0（28模式检测）
5. 检查 $代币 格式
6. POST 发布到币安广场
7. 记录到 memory/YYYY-MM-DD.md
```

### 文章类型分配（每日10-15篇）
| 类型 | 篇数 | 代币 |
|------|------|------|
| 行情分析/反差 | 2-3篇 | $BTC $ETH |
| 贵金属联动分析 | 2篇 | $XAU $XAG |
| 数据洞察（信息差）| 2篇 | 当日热门代币 |
| AI×交易心得 | 2篇 | $BTC |
| 操作记录/复盘 | 2篇 | 当日涨跌幅最大代币 |
| 活动/空投攻略 | 1篇 | 活动相关代币 |
| 争议观点 | 1篇 | 热度最高代币 |

### humanizer-cn v3.0 接入
每篇文章生成后必须检查并处理：
- 28个AI写作模式（见 humanizer-cn/SKILL.md）
- 代币格式标准化（裸BTC → ` $BTC `）
- 无邪禁用词清零
- 结尾必须是问句或留白
