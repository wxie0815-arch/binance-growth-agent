# 🦞 我的币安人生

## 项目定位
每个人在币安的交易历史，都是一面镜子。
「我的币安人生」帮你看清这面镜子，然后改变它。

## 5大功能
1. 🪞 **交易者人格分析**：读取现货+合约记录，识别风险偏好类型
2. 🗺️ **个性化策略建议**：按类型定制仓位、止损、品种选择
3. 📊 **每日交易追踪**：模拟盘或实盘，每日报告策略执行情况
4. 📅 **7日广场挑战**（可选）：每天在广场公开分享成长历程
5. 📨 **每日汇总推送**：北京时间22:00自动发送当日报告

## 数据模式
- **模拟盘**：testnet.binance.vision（无需API Key）
- **实盘**：用户提供只读API Key

## 风险偏好类型
| 类型 | 特征 | 核心建议 |
|------|------|---------|
| 🛡️ 保守型 | 低频、低杠杆、持仓稳 | Earn理财为主 |
| ⚖️ 稳健型 | 有纪律、止损好 | 趋势跟踪+Launchpool |
| 🎯 平衡型 | 现货合约均有 | 聚焦2-3主赛道 |
| 🚀 进取型 | 高频、追热点 | 强制止损+交易次数上限 |
| ⚡ 激进型 | 高杠杆、情绪化 | 立即降杠杆+每日复盘 |

## 子Agent架构
```
主Agent（我的币安人生）
├── 🪞 风险镜子    → 人格分析报告
├── 🗺️ 收益地图    → 最优配置建议
├── 📊 合约复盘教练 → 亏损拆解+错误识别
└── 🎓 苏格拉底    → 情景训练+思维改变
```
